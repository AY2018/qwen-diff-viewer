五月二十五日（庚申四月初八日癸未） 火曜日（即星期二） 民國九年日記

道徳為保護自由之本 （司美士）

| 提 交 | 要 際 |
| :--- | :--- |
| 聞大外姑喪 | 張子武歸 決計 |
| 電 | 西林、正明、吳、葛、趙電。趙、魯、蕭。發方家、紫雲、契隽、八一。得百卅五、卅七、淑三九、紫雲、契隽。 |
| 候氣 | 晴 |
| 度溫 | 八十の度 |

五時醒，坐三十分起。與呂言事。讀英文，盡冬夜故事一則。粥後，浴。
凡來，言辭不與席事，只合聽之。醉六來商事。子武亦到，相與論大事，數語而決。蓋同一失信，無久暫之分，而有利害之別，盡人聽天而已。梅齋來。（賴錦文來。）午飯後，作書，發電，說電話。晚，具酒食，請子武、賴錦文、松邨、梅齋、詠樵、彭青岑、奇峯、子理、毓昆。既散，與蕭、趙電話。子武久談。松邨、賴錦文來談。夜與張、呂談。至二時乃睡。
浴後愈熱。

決計

解是雨花新浴佛諸天誰供洗兒錢（楊顯祖）
