# 九月七日

**天氣** 陰雨 豫 記

**發信**
廿三書
電子武、靜生

**受信**
子武電、道士
歐二信

三時起，衣大禮服。乘馬至府學宮，行祀孔子禮，
舊所謂丁祭也。文官畢集，禮如舊章，但鞠躬不
拜耳。禮畢，已天明，遂歸。治文書，作家書上母
親，以昨日得諭也。飯後，閱稿件畢，書挽蕭禮衡父
聯，代呂滿撰一聯。桂濤來談公事。仇楞生來。至政務
廳看辦公廳部署，尚未就緒。至軍署會食
後，劉鴻逵來見。一時，遂至會客廳，客殊少，看報
久之。至三時，入軍署辦公廳，治公事至五時散。（崑濤來，余琨來。）
今日殊熱，軍服尤悶極，歸而就浴，浴罷飯。頌雲
來，夔生來，陳坤載來，嵩生來。余、陳去，二程
談至二時乃去。頗覺倦，遂就寢。
連日衣軍服，鬱熱，皮膚起疿，廿年所未有也。
