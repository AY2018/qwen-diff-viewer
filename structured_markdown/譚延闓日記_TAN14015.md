# 十一月四日

**天氣** 晴大風

**記**

由西直門至三家店、黃邨，二等八角。由黃村至龍王堂，每驢三角。
由龍王堂僱轎，每日三元。宿龍王堂二日四餐，十六元。
飯碧雲戒壇，各六元。宿岫雲寺一日二餐，十四元。

早七時起。大風。飯後，偕同人輿往戒臺寺，（賃三驢載行李，乘僕人。過靈光寺、長安寺。）過康修親王園寢，下
輿，讀碑。過十景山，孤峯聳立，頗似金焦。渡渾河，水從冰下流，浩浩有
聲。凡二十餘里，乃至山下。磴道盤紆，山石歷落。輿人雖四，登陟殊
艱，歷四十分鐘乃至戒臺寺。山門侍者出迎，導觀諸勝，有自
在松、抱塔松、臥龍松。以九龍松為奇，實乃栝也，俗謂白皮松。惜
活動松已死去，惟乾隆題詩石在，蓋光緒中恭忠親王重建此
寺，誤傷松根所致云。登藏經閣，極高，然已東傾，終必速圯。至戒
壇，三丹重飾，絕巨麗，號為天下第一。有羅漢堂，列像五百，皆柟木
雕刻。（有遼碑一，大康十年。）和尚延客，飯於別室，蔬食麫飯甚精。三時起。出越羅睺
嶺，往潭柘嶺，絕高，行人盤旋，若蟻緣壁。既下，石徑極錯落難行，
至潭柘山則整飭。逕至岫雲寺，門外兩松相鈎連如門，然寺亦光緒中重修，
為殿五重，最高及山半。殿旁有白果樹，俗謂帝王樹，又有娑羅樹。二佛座
旁兩小龕，所奉為大青、小青，皆不得見，當已蟄矣。閣下有元妙嚴公主拜塼，
膝痕穿石，又念珠大如鴿卵。晚，宿猗玕亭後，亭中刻龍首，可引泉流
觴。晚聞歌聲，則留音機也。飯蔬食，不設飲。兩僧來談。
