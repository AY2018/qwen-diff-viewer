# 九月二十日

**天氣**：陰

**發信**：徐師、汪九、黎九、黎六、冬生

九時起。睡遂亘十二小時，甚不可矣。食粥。補昨日未完一紙，臨《麻姑》二紙。李筠厂來，久談去。大武亦出就診。為盧鴻滄作書，上徐菊翁，言漢冶萍事，交其携去。午飯，飲一巡，大武已戒酒矣。晚，有滷子麫，尚佳，小飲下之。讀《晉書》，盡一卷。為諸女講史事。十時寢，始蓋厚棉被。

鄧騫說甘卓，隗囂斷隴右，竇融兼河西，各據一方，鼎足之勢，故得文服天子，從容顧望。文服猶言文致，徒以虛文服也。

卞壺父粹，兄弟六人均登宰輔，世稱「卞氏六龍，玄仁無雙」。玄仁，粹字。

成帝幸王導宅堂，拜導婦，孔垣表不宜拜。導聞之曰，王茂弘駑疴耳，若卞望之之巖巖，刁玄亮之察察，戴若思之峰岠，當敢爾耶。

阮孚謂壺，卿恆無閑泰，常如含瓦石，不亦勞乎。
